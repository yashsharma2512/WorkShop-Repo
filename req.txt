Method 1: Using PowerShell (Requires Admin)
Open the Start menu, search for PowerShell, right-click it, and select Run as Administrator.
To check your current policy, type: Get-ExecutionPolicy -List.
To enable scripts for all users, type: Set-ExecutionPolicy RemoteSigned -Scope LocalMachine.
To enable scripts only for yourself (no admin rights needed), type: Set-ExecutionPolicy RemoteSigned -Scope CurrentUser.
Type Y or A and press Enter when prompted to confirm.
